📚 Academic Record Manager (Streamlit App)
📝 Overview

The Academic Record Manager is a simple and interactive web application built using Streamlit. It allows users to input, manage, and analyze student academic records efficiently. The app calculates total marks and percentage automatically and displays all records in a structured format.

🚀 Features
Add student details (Name, Roll Number)
Input subject-wise marks (Maths, Science, English, Computer)
Automatic calculation of:
Total Marks
Percentage
Display records in a table format
Download records as CSV file
Simple and user-friendly interface
🛠️ Technologies Used
Python
Streamlit
Pandas
▶️ How to Run the Project
1. Clone the Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2. Install Dependencies
pip install streamlit pandas
3. Run the Application
streamlit run app.py
